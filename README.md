# My Portfolio

This repository contains a lightweight personal portfolio site and the assets used by it.

- **Main page:** [index.html](../index.html)
- **Certificates folder:** [certificates](../certificates/)
- **Resume folder:** [resume](../resume/)

- **Deployment:** https://basavaraj607.vercel.app
- **Source (GitHub):** https://github.com/Basavaraj607/My_Portfolio

Quick notes
- To view the site locally (recommended):

		Then open in your browser and navigate to `index.html`.

- Files in the `certificates` folder are linked from the Certifications section and open in a new tab for viewing.
- The resume is available in the `resume` folder and the site provides both a preview and a download option.
